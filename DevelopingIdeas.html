Design Specifications

There are some essential factors that make a website good, including some ergonomic factors such as navigation, easy access to help, and aesthetics to make the user enjoy their experience.

Success Criteria:
  • Must have a logo on the top at all times which returns you to the home page in case the user gets lost.
  • Must turn JSON data into more useable data such as graphs and tables.
  • Must have interactive buttons as well as pulldowns which react on click
  • Must include dropdown box with countries and search bar for countries.
  • The website must be neat and organized in a way which is pleasing to the eye, but is also easy navigatable.
  • An obvious target audience must be identified. This way we can target the age group and make our website more appealinge to an age group.
  • Must have an info/frequently asked questions button
